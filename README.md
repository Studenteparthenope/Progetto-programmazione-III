# Progetto-programmazione-III
L'applicazione "La ricetta per te" è una piattaforma sviluppata in Java (ambiente Eclipse) che permette la gestione e l'esplorazione di ricette. L'interfaccia utente è stata realizzata utilizzando le librerie Swing e AWT. Il sistema implementa i Design Pattern Factory Method, Memento, State e Strategy per garantire modularità e robustezza.
